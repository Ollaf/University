	ArmyList
Моделирование боя нескольких армий(на основе списка).
Бой продолжается до тех пор, пока агрессия армий не станет = 0, либо пока армия не погибнет.
Вывод результатов боя в HTML файл(по раундам).
Все параметры армий(численность, сила солдат, здоровье) задаются случайно.
(Находится на стадии доработки)

	AbsFactory
Реальзация паттерна "Абстрактная фабрика" на примере "Солдат" и "Ученых".

	Factory
Реальзация паттерна "Фабричный метод" на примере "Собак" и "Кошек".

	Gauss_Jordan
Численное решение СЛАУ методом Жордана-Гаусса.

	Anagrams
Ищем в файле анаграммы для введенного слова.

	List
Реализация списка на основе указателей.

	Ising_Model-2DH
Моделирование критического поведения двухмерной однородной модели Изинга вблизи критической температуры.
Алгоритм Метрополиса.
